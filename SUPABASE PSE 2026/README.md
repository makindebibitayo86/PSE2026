# My First Website

This is my very first website project! I'm learning HTML, CSS, and JavaScript.

## What's in this project:

- `index.html` - The main web page
- `styles/style.css` - Makes the page look nice
- `scripts/script.js` - Adds interactivity
- `assets/` - For images and other files (empty for now)

## How to view the website:

1. Open `index.html` in your web browser
2. Or use a local server (like Live Server in VS Code)

- JavaScript: Making pages interactive
